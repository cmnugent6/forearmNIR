This code provides the implementation of the forearm_NIR datset.

Tihs data set was a quired at UNCC by Michael Nugent and Matthew Sanchez.


-----------------------------------------------------------------------------

The github contains the different approaches for utilizing this data.

These approaches consist of yolov8, and UNET to train models for vein segmentation.

The overall setup of the images can be found in preprocessing.

These images work best when processed to exaggerate the veins.

to learn more about our data set, view our report.
